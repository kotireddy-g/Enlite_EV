## Project Overview

**Enlite EV Care** is a professional Electric Vehicle (EV) service center web application built for showcasing EV maintenance and repair services for 2-wheelers, 3-wheelers, and buses in Hyderabad. The application is a fully functional, modern, and responsive full-stack web application — production-ready and live on Hostinger.

---

## Scope of Work Delivered

### 1. 🖥️ Frontend Development
- Built a modern, responsive **React 18 + TypeScript** web application
- Implemented mobile-first responsive design using **Tailwind CSS**
- Used **shadcn/ui** component library for consistent, accessible UI components
- Developed all key sections:
  - Landing / Hero Section
  - Services Section (2-wheelers, 3-wheelers, Buses)
  - Contact / Enquiry Section
  - Admin Login & Dashboard
- Smooth scrolling navigation, glass-morphism effects, and gradient styling
- Toast notifications and form validation using **React Hook Form + Zod**
- Used **TanStack Query** for efficient data fetching and caching

### 2. ⚙️ Backend Development
- Built a **Node.js + Express.js** REST API server with **TypeScript**
- Developed full API endpoints for:
  - Contact/Enquiry form submissions (`/api/contact`)
  - Admin authentication and session management
  - Service booking management (CRUD operations)
  - Admin dashboard data (query listing, status updates)
- Custom **logging middleware** for request/response tracking
- **Error handling middleware** with standardized JSON responses
- **Session management** with secure passport-based authentication

### 3. 🗄️ Database Setup
- Configured **PostgreSQL** database using **Neon Database** (cloud-hosted)
- Designed schema using **Drizzle ORM** with type-safe queries:
  - `contacts` table — Stores all customer enquiries with status tracking
  - `users` table — Admin user authentication and management
- Integrated **Zod schema validation** with Drizzle for type-safe operations

### 4. 🚀 Deployment — Hostinger
- Built and deployed the full-stack application on **Hostinger** shared hosting
- Configured `.htaccess` for proper routing of the React (SPA) application
- Set up production build pipeline using **Vite + esbuild**
- Ensured the server and client are both live and accessible

### 5. 📧 Email Configuration
- Configured business email for the client on **Hostinger**
- Set up email routing and DNS records (MX, SPF, DKIM) for reliable email delivery
- Verified email functionality end-to-end
